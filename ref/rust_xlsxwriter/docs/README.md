# rust_xlsxwriter 项目架构文档

本文档提供了 rust_xlsxwriter 项目的架构概览和模块组织结构。

## 项目简介

`rust_xlsxwriter` 是一个用于编写 Excel 2007+ xlsx 格式文件的 Rust 库。该库专注于性能和与 Excel 文件格式的一致性，不支持读取或修改现有文件。

### 设计原则

- **格式保真度**：输出与 Excel 生成的格式高度一致
- **高性能**：接近 C 语言版本的性能，支持常量内存模式
- **完善文档**：提供详尽的 API 文档、用户指南和示例
- **功能丰富**：支持广泛的 Excel 功能
- **只写设计**：专注于写入操作
- **跨语言家族**：与 C、Python、Perl 版本共享设计和改进

### 技术特点

- 纯 Rust 实现
- 使用自定义 XML 写入器确保格式一致性
- 超过 1000 个测试用例
- 支持常量内存模式处理大文件
- 零安全保证
