# excel-toolset 与 OfficeCLI 功能对比分析

## 一、项目概览

| 维度 | excel-toolset | OfficeCLI |
|------|--------------|-----------|
| **语言** | Rust（Cargo Workspace） | C#（.NET 10, PublishSingleFile） |
| **定位** | Excel 专用操作工具集 | Office 全格式 CLI 工具（Word/Excel/PPT） |
| **目标用户** | AI Agent / LLM | AI Agent |
| **许可证** | MIT | Apache 2.0 |
| **安装方式** | Cargo 编译 | 单二进制下载 / Homebrew / npm |
| **接口形态** | CLI + HTTP REST API | CLI + MCP Server（JSON-RPC） |
| **Excel 写入引擎** | calamine（读）+ rust_xlsxwriter（写） | DocumentFormat.OpenXml |
| **公式引擎** | 依赖追踪/语义解释（非求值） | 350+ 函数求值引擎 |
| **SQL 支持** | DuckDB 嵌入式 SQL | 无（通过查询选择器替代） |
| **多语言 SDK** | 无 | Python + Node.js |
| **实时预览** | 无 | watch 命令 + HTTP SSE |

## 三、excel-toolset 相对 OfficeCLI 的独特优势

| 优势 | 说明 |
|------|------|
| **HTTP REST API** | 完整的 60+ REST 端点，天然适配微服务架构和 Web 集成 |
| **DuckDB SQL 查询** | 可用完整 SQL 语法查询 Excel 数据，支持跨表 JOIN、聚合分析 |
| **公式依赖追踪** | 分析公式间的引用关系，区分主动变更与被动变更 |
| **公式语义解释** | 将复杂公式翻译为自然语言说明，适合 AI Agent 理解 |
| **文件级 Diff** | 完整的 Excel 差异对比引擎，含自然语言输出 |
| **Git diff driver** | 可直接集成到 Git 工作流，追踪 .xlsx 变更 |
| **安全链路** | 指纹→备份→dry-run→执行→回滚 的完整安全保护 |
| **自动附属 Diff** | 每次写操作自动返回变更详情 |
| **集成测试完善** | 每个 crate 均有独立测试，e2e 测试覆盖主要工作流 |
| **Rust 生态** | 高性能、零运行时、编译期安全保障 |

## 五、架构差异分析

| 方面 | excel-toolset | OfficeCLI |
|------|--------------|-----------|
| **代码组织** | Cargo Workspace 多 crate 分层：types → core → sql/diff → cli/http | 单项目 partial class 划分：Core/ + Handlers/{Excel,Word,Pptx}/ |
| **命令设计** | 原子化操作：一个命令 = 一个 Excel 动作 | 三层架构：view（读）→ get/set/add/remove（DOM）→ raw（XML） |
| **参数结构** | CLI、HTTP、底层函数共用同一套 DTO | CLI 参数直接映射到方法调用 |
| **写入方式** | 读原文件→内存修改→新建文件→覆盖 | 直接操作 OpenXml DOM |
| **扩展性** | HTTP API 优先，天然微服务化 | 插件系统 + MCP 标准协议 |

## 六、总结建议

excel-toolset 在 **HTTP API 生态、SQL 查询、公式智能化分析、文件 Diff 对比、操作安全链路** 等方面具有独特优势，定位清晰。

对照 OfficeCLI，建议优先补充以下关键方向：

1. **数据透视表** -- Excel 最核心的高级数据分析能力，是最大功能空白
2. **数据验证与智能表格** -- 企业级 Excel 的标配功能
3. **MCP Server 集成** -- 与 AI Agent 生态的标准对接方式
4. **条件格式与图表增强** -- 提升数据可视化能力
5. **SDK 与预览** -- 降低用户接入门槛，提升交互体验

这些补充将显著增强 excel-toolset 在 AI Agent 场景下的竞争力，同时保持其在 HTTP API、SQL 查询、Diff 对比等方面的差异化优势。
