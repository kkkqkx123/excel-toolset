## 核心工具实现逻辑

### 1. 数据读取工具 (read_range)

#### 功能特性

- **三种读取模式**：
  - `compact`: 默认模式，返回Markdown表格，token高效
  - `csv`: 返回CSV格式，适合数据分析
  - `detailed`: 返回完整JSON，包含公式和格式信息

#### 实现逻辑

```typescript
// 1. 参数验证
const { range, mode = 'compact' } = params;

// 2. 解析范围地址
const { sheetName, address } = qualifiedAddress(range);

// 3. 执行Office.js操作
const rangeObj = sheet.getRange(address);
rangeObj.load('values,formulas,numberFormat,rowCount,columnCount');
await context.sync();

// 4. 根据模式格式化输出
switch (mode) {
  case 'compact':
    return formatAsMarkdownTable(values);
  case 'csv':
    return convertToCsv(values);
  case 'detailed':
    return buildDetailedJson(values, formulas, numberFormat);
}
```

#### 格式化特性

- **表格渲染**：包含列字母（A, B, C...）和行号
- **空范围处理**：返回"_All cells are empty._"而非空白表
- **错误标记**：自动检测并标记错误单元格
- **格式人性化**：已知格式字符串显示人类可读标签

### 2. 数据写入工具 (write_cells)

#### 功能特性

- **覆盖保护**：默认阻止覆盖现有数据
- **公式支持**：字符串以"="开头识别为公式
- **自动验证**：写入后读取验证，报告公式错误
- **变更追踪**：记录写入前后的变化

#### 实现逻辑

```typescript
// 1. 覆盖检查
if (!allow_overwrite) {
  const existingValues = await readExistingValues(range);
  if (hasData(existingValues)) {
    return { blocked: true, existingData };
  }
}

// 2. 执行写入操作
const rangeObj = sheet.getRange(address);
rangeObj.values = values;

// 3. 特殊处理公式
for (const cell of cells) {
  if (typeof cell.value === 'string' && cell.value.startsWith('=')) {
    cell.formula = cell.value;
    cell.value = null;
  }
}

// 4. 同步并验证
await context.sync();
const writtenValues = rangeObj.values;
const errors = findErrors(writtenValues);

// 5. 创建快照（如果需要）
if (shouldCreateSnapshot(writtenValues)) {
  await createPreMutationSnapshot();
}

// 6. 记录变更
const changes = buildCellChangeSummary(before, after);
await appendToAuditLog(changes);
```

#### 安全机制

- **快照大小限制**：超过`MAX_RECOVERY_CELLS`时跳过快照
- **公式验证**：使用`validateFormula()`检查语法
- **错误检测**：自动检测并报告公式错误
- **回滚支持**：通过快照支持操作回滚

### 3. 公式填充工具 (fill_formula)

#### 功能特性

- **避免大数组**：使用Excel自动填充而非构造大型2D数组
- **引用调整**：相对引用自动调整
- **覆盖保护**：与write_cells相同的保护机制

#### 实现逻辑

```typescript
// 1. 验证公式
validateFormula(formula);

// 2. 检查覆盖
if (!allow_overwrite && hasData(range)) {
  return { blocked: true };
}

// 3. 设置左上角单元格
const topLeft = range.getCell(0, 0);
topLeft.formula = formula;

// 4. 自动填充到整个范围
const entireRange = worksheet.getRange(rangeAddress);
topLeft.autoFill(entireRange, Excel.AutoFillType.fillDefault);

// 5. 同步并验证
await context.sync();
const errors = findErrors(entireRange.values);
```

#### 优势对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| 2D数组写入 | 精确控制 | 内存占用大，引用需要手动调整 |
| 自动填充 | 内存高效，引用自动调整 | 只适合单一公式模式 |

### 4. 格式化工具 (format_cells)

#### 功能特性

- **预设样式**：number, integer, currency, percent等
- **结构样式**：header, total-row, subtotal等
- **样式组合**：支持样式数组，左到右合并
- **个别参数覆盖**：直接参数优先于样式属性

#### 实现逻辑

```typescript
// 1. 解析样式参数
const resolvedStyles = resolveStyles({
  style: styleParams,        // 预设样式
  bold: boldParam,           // 直接参数
  font_color: fontColorParam,
  // ... 其他参数
});

// 2. 解析范围（支持多范围）
const ranges = parseMultiRange(rangeAddress);  // "A1:B10,D1:F10"

// 3. 应用格式
for (const range of ranges) {
  // 字体格式
  range.format.font.name = resolvedStyles.fontName;
  range.format.font.size = resolvedStyles.fontSize;
  range.format.font.bold = resolvedStyles.bold;
  range.format.font.color = resolvedStyles.fontColor;

  // 填充格式
  range.format.fill.color = resolvedStyles.fillColor;

  // 数字格式
  range.numberFormat = resolvedStyles.numberFormat;

  // 对齐格式
  range.format.horizontalAlignment = resolvedStyles.horizontalAlignment;
  range.format.verticalAlignment = resolvedStyles.verticalAlignment;

  // 边框格式
  applyBorders(range, resolvedStyles.borders);

  // 列宽和行高
  if (resolvedStyles.columnWidth) {
    range.getEntireColumn().columnWidth = resolvedStyles.columnWidth;
  }
  if (resolvedStyles.rowHeight) {
    range.getEntireRow().rowHeight = resolvedStyles.rowHeight;
  }

  // 合并单元格
  if (resolvedStyles.merge) {
    range.merge();
  }
}

// 4. 同步
await context.sync();
```

#### 样式解析

```typescript
function resolveStyles(params) {
  let styles = { ...DEFAULT_STYLES };

  // 1. 应用预设样式
  if (params.style) {
    const styleArray = Array.isArray(params.style) ? params.style : [params.style];
    for (const styleName of styleArray) {
      const preset = PRESET_STYLES[styleName];
      if (preset) {
        styles = { ...styles, ...preset };
      }
    }
  }

  // 2. 直接参数覆盖
  return {
    ...styles,
    ...pickDefined(params, STYLE_PARAMS)
  };
}
```

#### 边框处理

```typescript
function applyBorders(range, borders) {
  const borderStyles = {
    'none': 'None',
    'thin': 'Continuous',
    'medium': 'Continuous',
    'thick': 'Continuous'
  };

  const borderWeights = {
    'thin': 'Thin',
    'medium': 'Medium',
    'thick': 'Thick'
  };

  for (const edge of ['top', 'bottom', 'left', 'right']) {
    const borderStyle = borders[edge] || borders.all;
    if (borderStyle && borderStyle !== 'none') {
      const border = range.format.borders.getItem(edge);
      border.style = borderStyles[borderStyle];
      border.weight = borderWeights[borderStyle];
    }
  }
}
```

### 5. 搜索工具 (search_workbook)

#### 功能特性

- **搜索模式**：值搜索、公式搜索
- **正则支持**：支持正则表达式搜索
- **上下文返回**：可选返回匹配行周围的上下文
- **分页支持**：支持offset和max_results参数

#### 实现逻辑

```typescript
// 1. 确定搜索范围
const sheetsToSearch = sheet
  ? [workbook.worksheets.getItem(sheet)]
  : workbook.worksheets;

// 2. 构建搜索模式
const searchPattern = use_regex
  ? new RegExp(query, 'i')
  : new RegExp(escapeRegex(query), 'i');

// 3. 遍历工作表
for (const sheet of sheetsToSearch) {
  const usedRange = sheet.getUsedRange();

  if (search_formulas) {
    // 搜索公式
    usedRange.load('formulas');
    await context.sync();

    for (let row = 0; row < usedRange.rowCount; row++) {
      for (let col = 0; col < usedRange.columnCount; col++) {
        const formula = usedRange.formulas[row][col];
        if (formula && searchPattern.test(formula)) {
          matches.push({ sheet, row, col, formula });
        }
      }
    }
  } else {
    // 搜索值
    usedRange.load('values');
    await context.sync();

    for (let row = 0; row < usedRange.rowCount; row++) {
      for (let col = 0; col < usedRange.columnCount; col++) {
        const value = usedRange.values[row][col];
        if (value != null && searchPattern.test(String(value))) {
          matches.push({ sheet, row, col, value });
        }
      }
    }
  }
}

// 4. 应用上下文
if (context_rows > 0) {
  for (const match of matches) {
    match.context = extractContext(match, context_rows);
  }
}

// 5. 应用分页
const paginatedMatches = matches
  .slice(offset, offset + max_results);
```

### 6. 依赖追踪工具 (trace_dependencies)

#### 功能特性

- **两种模式**：
  - `precedents`: 追踪前驱（输入来源）
  - `dependents`: 追踪后继（影响范围）

#### 实现逻辑

```typescript
// 1. 获取目标单元格
const targetCell = worksheet.getRange(cellAddress);
targetCell.load('formula');
await context.sync();

// 2. 检查是否为公式
if (!targetCell.formula || !targetCell.formula.startsWith('=')) {
  return { isFormula: false, value: targetCell.values[0][0] };
}

// 3. 根据模式追踪
switch (mode) {
  case 'precedents':
    return await tracePrecedents(targetCell);
  case 'dependents':
    return await traceDependents(targetCell);
}

async function tracePrecedents(cell) {
  // 尝试使用Office.js API
  const precedents = cell.getDirectPrecedents();
  precedents.load('areas');
  await context.sync();

  if (!precedents.isNullObject) {
    return buildTreeFromAreas(precedents.areas);
  }

  // 降级到公式解析
  return parseFormulaForPrecedents(cell.formula);
}
```

### 7. 公式解释工具 (explain_formula)

#### 功能特性

- **自然语言解释**：提供公式的可读性解释
- **引用预览**：显示直接引用的值
- **当前值展示**：显示公式的计算结果
- **错误处理**：对非公式单元格返回静态值解释

#### 实现逻辑

```typescript
// 1. 获取单元格信息
const cell = worksheet.getRange(cellAddress);
cell.load('formula,values');
await context.sync();

// 2. 检查是否为公式
const formula = cell.formula?.[0]?.[0];
if (!formula || !formula.startsWith('=')) {
  return {
    type: 'static_value',
    value: cell.values[0][0],
    explanation: `单元格包含静态值：${cell.values[0][0]}`
  };
}

// 3. 解析公式引用
const references = extractReferences(formula);

// 4. 限制引用数量（性能考虑）
const limitedRefs = references.slice(0, max_references);

// 5. 获取引用值
const refValues = {};
for (const ref of limitedRefs) {
  const refRange = worksheet.getRange(ref);
  refRange.load('values');
  await context.sync();
  refValues[ref] = refRange.values;
}

// 6. 生成解释
const explanation = await generateExplanation(formula, refValues);

return {
  type: 'formula',
  formula: formula,
  currentValue: cell.values[0][0],
  explanation: explanation,
  references: refValues
};
```

## 实验性工具实现

### 1. Python执行工具 (python_run)

#### 双模式执行

**Pyodide模式**（默认）
- 浏览器内WebAssembly执行
- 零配置，自动安装标准库
- 冷启动~15MB，浏览器缓存

**本地桥接模式**
- 连接到本地Python进程
- 支持C扩展和文件系统
- 需要用户确认和配置

#### 实现逻辑

```typescript
// 1. 确定执行模式
const effectiveBridgeUrl = getEffectiveBridgeUrl();
const useBridge = effectiveBridgeUrl && await isBridgeReachable(effectiveBridgeUrl);

// 2. 用户确认（仅桥接模式）
if (useBridge && !isConfirmedForBridge(effectiveBridgeUrl)) {
  return { requiresConfirmation: true };
}

// 3. 执行Python代码
let result;
if (useBridge) {
  result = await executeViaBridge(effectiveBridgeUrl, code, inputJson);
} else {
  result = await executeViaPyodide(code, inputJson);
}

// 4. 处理结果
return {
  stdout: result.stdout,
  stderr: result.stderr,
  result: result.json_output,
  executionMode: useBridge ? 'bridge' : 'pyodide'
};
```

#### 实现逻辑

```typescript
// 1. 加载MCP服务器配置
const servers = await loadMcpServers();

// 2. 处理不同操作
switch (operation) {
  case 'list':
    return listAllServersAndTools(servers);

  case 'search':
    return searchTools(query, servers);

  case 'describe':
    return describeTool(toolName, servers);

  case 'connect':
    const server = servers[serverName];
    return connectToServer(server);

  case 'call':
    return callTool(toolName, args, servers);
}

// 3. 调用工具示例
async function callTool(toolName, args, servers) {
  const server = findServerForTool(toolName, servers);
  const response = await fetch(`${server.url}/tools/call`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${server.token}`
    },
    body: JSON.stringify({
      tool: toolName,
      arguments: JSON.parse(args)
    })
  });

  return await response.json();
}
```

```typescript
async function installCodeExtension(name, code, replaceExisting) {
  // 1. 检查现有扩展
  const existing = getExtension(name);
  if (existing && !replaceExisting) {
    return { error: 'Extension already exists' };
  }

  // 2. 验证代码
  const validationResult = validateExtensionCode(code);
  if (!validationResult.valid) {
    return { error: validationResult.reason };
  }

  // 3. 移除现有扩展
  if (existing) {
    await unloadExtension(name);

  // 4. 安装新扩展
  const extension = await loadExtension(name, code);

  // 5. 注册扩展工具
  for (const tool of extension.tools) {
    registerTool(tool);
  }

  return { success: true, extension };
}
```

```typescript
// 执行工具前
async function executeWithWorkbookCoordinator(tool, params) {
  // 1. 获取工作簿锁
  const lock = await acquireWorkbookLock(workbookId);

  try {
    // 2. 检查蓝图有效性
    if (tool.policy === MUTATE_STRUCTURE) {
      invalidateBlueprint();
    }

    // 3. 执行工具
    const result = await tool.execute(toolId, params);

    // 4. 处理变更
    if (tool.policy !== READ_NONE) {
      await handleWorkbookChanges(result);
    }

    return result;
  } finally {
    // 5. 释放锁
    await releaseWorkbookLock(lock);
  }
}
```

### 3. 输出截断

```typescript
// 全局输出截断包装器
function withOutputTruncation(tool) {
  return async (toolId, params) => {
    const result = await tool.execute(toolId, params);

    // 检查是否需要截断
    const text = result.content[0]?.text || '';
    const size = new TextEncoder().encode(text).length;
    const lines = text.split('\n').length;

    if (size > MAX_SIZE || lines > MAX_LINES) {
      // 确定截断策略
      const strategy = determineTruncationStrategy(tool.name);

      // 执行截断
      const truncated = truncateText(text, strategy);

      // 保存完整输出（如果可能）
      await saveFullOutput(toolId, text);

      // 返回截断结果
      return {
        ...result,
        content: [{ type: 'text', text: truncated }],
        details: {
          outputTruncation: {
            strategy,
            hitReason: size > MAX_SIZE ? 'size' : 'lines',
            totalSize: size,
            outputSize: new TextEncoder().encode(truncated).length,
            totalLines: lines,
            outputLines: truncated.split('\n').length,
            maxSize: MAX_SIZE,
            maxLines: MAX_LINES
          }
        }
      };
    }

    return result;
  };
}
```

```typescript
async function executeWithErrorRecovery(tool, params) {
  try {
    return await tool.execute(toolId, params);
  } catch (error) {
    // 1. 记录错误
    logError(error);

    // 2. 尝试恢复
    if (isRecoverable(error)) {
      const recovery = await attemptRecovery(error);
      if (recovery.success) {
        return recovery.result;
      }
    }

    // 3. 返回结构化错误
    return {
      ok: false,
      content: [{ type: 'text', text: error.message }],
      details: buildErrorDetails(error)
    };
  }
}
```
