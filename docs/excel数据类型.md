# Excel常用数据类型及区分方法

## 一、Excel六大基础数据类型

| 数据类型                | 说明                                                     | 示例                                                               |
| ----------------------- | -------------------------------------------------------- | ------------------------------------------------------------------ |
| **数值型（数字）**      | 可算术运算，包含整数、小数、日期、时间、百分比、科学计数 | `123`、`3.14`、`2025/1/1`、`9:30`、`50%`                           |
| **文本型（字符串）**    | 纯文字、不能直接算数，数字当成文本存储                   | `张三`、`00101`、`"123"`                                           |
| **逻辑型（布尔）**      | 只有两个固定值：TRUE（真）、FALSE（假），多用于公式判断  | `TRUE`、`FALSE`                                                    |
| **错误值类型**          | 公式运算出错返回的标识，共7种常见错误码                  | `#N/A`、`#VALUE!`、`#DIV/0!`、`#NAME?`、`#NULL!`、`#NUM!`、`#REF!` |
| **空值（空白）**        | 单元格无任何内容，真空单元格                             | 空单元格（不是输入空格）                                           |
| **超链接/对象（拓展）** | 链接、批注、图片、控件、嵌入式文档（较少归类基础数据）   | 网址链接、插入表格                                                 |

> 补充：**日期、时间本质是特殊数值**，Excel用序列号存储日期，1代表1900-1-1。

## 二、快速区分单元格数据的4种方法

### 1. 看单元格默认对齐（最直观）

1. **数值/日期/时间：靠右对齐**
2. **文本：靠左对齐**
3. **逻辑值TRUE/FALSE、错误值：居中对齐**
   > 注意：手动修改对齐方式会失效，不能100%依靠。

### 2. 选中单元格 → 查看【开始】选项卡「数字格式」

- 常规/数值/日期/百分比 → **数值类**
- 文本 → **文本类型**
- 特殊（邮政编码、身份证）→ 本质文本

### 3. IS类函数精准判断（公式区分，最准确）

在空白单元格输入公式：

- `=ISNUMBER(A1)`：返回TRUE=**数值（含日期时间）**；FALSE=非数值
- `=ISTEXT(A1)`：返回TRUE=**文本**
- `=ISLOGICAL(A1)`：返回TRUE=**布尔值**
- `=ISERROR(A1)`：返回TRUE=**错误值**
- `=ISBLANK(A1)`：返回TRUE=**空单元格**

### 4. 编辑栏+输入特征区分

1. **文本数字特征**：单元格数字开头带英文单引号`'0001`，编辑栏可见单引号，无法求和；
2. **日期时间**：修改单元格格式为「常规」会变成一串数字（日期序列号）；
3. **空格文本**：单元格肉眼空白，但`ISBLANK`返回FALSE，属于带空格文本。

## 三、易混易错区分

1. **`123`(数值) vs `'123`(文本数字)**
   数值可加减乘除，文本数字求和会被当成0；
2. **真空单元格 vs 输入空格" "**
   真空：`ISBLANK=TRUE`；空格：属于文本，`ISBLANK=FALSE`；
3. **百分比50%**：底层是0.5（数值），改常规格式显示0.5。
