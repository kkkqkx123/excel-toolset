## 一、核心必做亮点（3个，项目立足之本，解决行业最大痛点）
这3个功能是用户刚需，也是你项目区别于所有普通Excel diff工具的核心，**必须首发实现**：

### 1. Git原生无缝集成：解决Excel二进制文件Git Diff乱码难题
✅ **解决的问题**：Git默认无法对比Excel（二进制文件），diff全是乱码，无法看版本差异；
✅ **实现方式**：
- 一行命令集成Git（`git config diff.excelvision.excel`）
- 自动拦截Git的Excel版本对比，替换为**结构化文本diff**
- 直接在`git diff` / `git log`中输出类CSV的极简Excel diff
✅ **亮点价值**：
  全球90%的开发者用Git管理Excel都踩坑，你的工具**让Excel在Git里和代码一样可diff**，这是开源项目的**杀手级功能**。

### 2. AI Agent + 人工操作 双源统一溯源（你的核心需求）
✅ **解决的问题**：市面上工具要么只支持人工diff，要么只记录AI日志，无法统一查看混合操作的历史；
✅ **实现方式**：
- 人工操作：通过Excel解析库自动生成结构化diff
- AI Agent操作：读取工具调用日志/JSON操作记录
- 合并为**统一操作历史**，标注「人工修改 / AI修改」
✅ **亮点价值**：
  业内**第一个同时兼容Git+人工+AI**的Excel版本管理工具，完美匹配你的AI Agent场景。

### 3. 智能公式降噪Diff：区分「主动修改」和「公式级联变化」
✅ **解决的问题**：普通Excel diff会把公式自动计算的被动变化全部标记，diff列表爆炸；
✅ **实现方式**：
- 自动识别公式依赖链
- **只高亮AI/人工主动修改的单元格**
- 公式级联变化折叠为灰色提示，不冗余展示
- 公式单元格同时展示 `公式+值` 双列diff
✅ **亮点价值**：
  比所有开源Excel diff工具更智能，**diff结果干净、可读，1秒看懂修改**。
